### 1주차 미션 - 숫자 야구

구현할 기능 목록

1. 숫자 랜덤 생성

- 1부터 9까지의 숫자
- 3자리수
- 서로 다른 수 (중복 x)

2. 사용자 입력

- 예외) 1-9의 숫자가 아닌 경우(0이나 음수/문자,실수 등)
- 예외) 3자리가 아닌 경우
- 예외) 중복된 수가 있는 경우
- 예외사항 발생할 경우 IllegalArgumentException 발생시킨 후 애플리케이션 종료

3. 랜덤 생성 숫자와 사용자 입력 비교

- 같은 수가 같은 자리 : 스트라이크
- 같은수가 다른 자리 : 볼
- 같은수가 전혀 없음 : 낫싱

4. 비교 결과 출력

- 낫싱일 경우 : 낫싱 출력
- 볼, 스트라이크 카운트 후 출력
- 출력순서 : (x)볼 (y)스트라이크
- y=3일 경우 게임 종료 문구 출력
- y는 3이 아닌 경우 2. 사용자입력 으로 전환

5. 종료 조건 입력

- 1 입력 재시작 : 1. 숫자랜덤으로 전환
- 2 입력 종료 : 게임 종료
- 예외) 1과 2가 아닌 잘못된 값을 입력한 경우 IllegalArgumentException 발생 시킨 후 종료
